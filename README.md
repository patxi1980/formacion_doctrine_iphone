Formacion Doctrine convertido a Symfony2
========================================

Ejemplos de foramción de Doctrine en Symfony2.

Listado de ejercicios

1) Listado de usuarios
----------------------

Listado de los primeros 20 usuarios.

2) Incidentes de un usuario
---------------------------

A traves de un nombre de usuario (username) se cogen los incidentes que ha declarado por ese usuario

3) Restauranets más votados
---------------------------

Buscar los restaurantes con más votos

4) Restaurantes con más incidentes
----------------------------------

Buscar los restaurantes con más incidentes de un tipo.

5) Amigos de un usuario
-----------------------

A traves de un nombre de usuario (username) se cogen todos los amigos de ese usuario.
